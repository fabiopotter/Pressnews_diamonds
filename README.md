# pressnews
Project for create a Press News Website

Roadmap:

1-Create a database with Script included in folder Database or restore .bak file
2-Change the name of connection in web.config to connection of user pc.
3-Check in Visual Studio if both projects are running at start
4-Set in WebForms project to start page "Login.aspx".
5-Run project, create user and go to Categories and News to CRUD.
